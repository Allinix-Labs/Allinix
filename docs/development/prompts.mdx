# Prompting in Allinix

Prompts are the foundation of how we communicate with and guide our AI agents. The efficiency and effectiveness of our agents heavily depend on how well we structure these prompts.

<Callout type="info">
  We primarily use two models in Allinix: - GPT-3.5-turbo - GPT-4 Both models
  are highly sensitive to prompt nuances, making proper prompt engineering
  crucial.
</Callout>

    ### Advanced Prompting Techniques #### Plan and Solve (PS) ```mermaid graph
    TD A[Problem Input] --> B[Understanding] B --> C[Variable Extraction] C -->
    D[Step-by-Step Plan] D --> E[Solution Execution] ```
    <Callout type="info">
      Plan and Solve enhances chain-of-thought prompting by breaking down
      complex problems into manageable steps. Learn more in the
      [Plan-and-Solve-Prompting
      repository](https://github.com/AGI-Edgerunners/Plan-and-Solve-Prompting).
    </Callout>
    #### ReAct (Reasoning + Action)
    <Card className="p-4 mt-4">
      <h4 className="font-semibold">ReAct Framework</h4>
      <p>
        Combines reasoning and action generation to create more coherent
        responses:
      </p>
      <ul className="list-disc pl-6">
        <li>Thought: Analyzes the current situation</li>
        <li>Action: Determines the next step</li>
        <li>Observation: Processes the result</li>
        <li>Response: Generates the final output</li>
      </ul>
    </Card>

## Best Practices

<Card>
  <div className="p-6 space-y-4">
    ### Tips for Effective Prompting 1. **Be Specific** - Clearly define
    expected output format - Include relevant context - Set explicit constraints
    2. **Iterate and Refine** - Test prompts with various inputs - Document
    successful patterns - Learn from unsuccessful attempts 3. **Consider
    Context** - Account for model limitations - Balance detail vs. conciseness -
    Maintain consistency across prompts
  </div>
</Card>
