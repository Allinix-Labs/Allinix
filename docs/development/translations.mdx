### Adding a New Language

<Steps>
  1. Update i18n Configuration
     ```javascript title="next/next-i18next.config.js"
     i18n: {
       defaultLocale: "en",
       locales: [
         "en",
         "hu",
         // Add your language code here
       ],
     }
     ```

2. Add Language Definition

   ```typescript title="next/src/utils/languages.ts"
   export const availableLanguages: Language[] = [
     ENGLISH,
     { code: "fr", name: "FranÃ§ais", flag: "ðŸ‡«ðŸ‡·" },
     // Add your language here
   ];
   ```

3. Create Translation Files
   ```bash
   mkdir next/public/locales/[language-code]
   # Add translations for all namespaces
   ```

  <Callout type="warning">
    Server restart may be required for new languages to take effect.
  </Callout>
</Steps>

## README Translation

<Steps>
  1. Create Translation File
     ```bash
     # In main/docs directory
     touch README.[language-code].md
     # Example: README.zh-HANS.md
     ```

2. Add Language Badge
   `markdown
      <a href="https://github.com/allinix/Allinix/blob/master/README.[language-code].md">
      <img src="https://img.shields.io/badge/lang-[Language]-blue.svg" alt="[Language]" />
     </a>
  `
   </Steps>
